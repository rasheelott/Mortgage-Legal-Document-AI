AI-Powered Mortgage Legal Document Classification
Project Overview

This graduate project demonstrates how machine learning and natural language processing can support mortgage servicing operations by automatically classifying county-recorded legal documents.

The prototype classifies:

Mechanic's Liens
Lis Pendens
Judgments
Tax Liens
Releases
Satisfactions
Assignments
Other Documents

using Logistic Regression and TF-IDF.

Technologies Used
Python
pandas
NumPy
scikit-learn
matplotlib
NLP
Machine Learning
Business Value

The prototype standardizes document review and estimates significant reductions in analyst review time while maintaining human oversight.
